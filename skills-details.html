Front End

Front end is what allows the user to interact with the website or application and creates the visuals that the user can see. Excellent front end development ensures that the site or application can work well across multiple devices or screen sizes.
Mastery:
  • HTML (HyperText Markup Language) : Creating the structure of my webpage
  • CSS (Cascading Style Sheets) : Styling my website and creating its aesthetic appeal
  • Javascript: Allowing my websites or applications to become interactive by enabling dynamic content such as animations, real-time updates, etc
  • React (Javascript Library) : Allows me to easily create reusable components throughout my site that efficiently updates and renders dynamic data
  • Bootstrap (CSS framework) : Allows me to create responsive and mobile friendly websites easily with the pre-built components that come with it

    • Back End

    Back end encompasses the backbone of every website or application. Essentially, it includes all the behind-the-scenes functionality that the user cannot see. It handles many tasks such as processing information, storing data, managing servers, communicating with APIs and ensuring that the user can communicate with the site/application.
    Mastery:
  • MySQL : Primarily use it as a database management system to store and structure data
  • Python : Makes data science very easy with the help of libraries such as Pandas and SciPy. Allows me to filter data, create data visualizations, and compute mathematical equations with ease
  • Flask (Python framework) Enables my web pages to be dynamic while handling HTTP requests and routing
  • PHP : Useful for connecting with databases such as MySQL, allows me to handle server-side operations and process data from forms
